# 🍲 CookBook App

Aplicación móvil desarrollada con React Native y Expo que permite explorar recetas por categorías como entradas, platos fuertes y postres.

## 🚀 Funcionalidades

* 📂 Navegación por categorías
* 🍽️ Visualización de recetas
* 📄 Pantalla de detalle con ingredientes y preparación
* 📱 Interfaz simple y amigable

## 🛠️ Tecnologías usadas

* React Native
* Expo
* React Navigation

## ▶️ Cómo ejecutar el proyecto

```bash
npm install
npx expo start
```
